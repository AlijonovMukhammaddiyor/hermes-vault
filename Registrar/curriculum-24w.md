# Curriculum — 24 Weeks (CS250 spine + CS301)

Rules: ≤4 tasks/day, 1 rest day/week (default Sun review counts as light). Exams end every EVEN
week (gate next unit, ≥C). Midterm W12, Finals W24. PD101 activates W9.

Daily ≤4-task budget is split across active courses by weight:
- **CS250 (weight 4)** — 1–2 problems/day + pattern note; spine below drives the unit each week.
- **CS301 (weight 4)** — ~1 design/week + a reading; units Fundamentals→Core→Case studies→Mocks
  advance ~1 unit/month (W1-4 Fundamentals, W5-8 Core, W9-16 Case studies, W17-24 Mocks).


Sun = quiz (odd wk) or exam (even wk), cumulative across active courses in scope.
The CS250 week-by-week below is the spine; CS301 rides the monthly cadence above.

## Month 1 — foundations
- W1 Arrays & Hashing (Easy→Med). Sun: quiz.
- W2 Two Pointers. Sun: **EXAM** (Arrays&Hashing + Two Pointers) → gate Sliding Window.
- W3 Sliding Window. Sun: quiz.
- W4 Stack. Sun: **EXAM** (cumulative) → gate Binary Search.

## Month 2
- W5 Binary Search. Sun: quiz.
- W6 Linked List. Sun: **EXAM** → gate Trees.
- W7 Trees. Sun: quiz.
- W8 Tries. Sun: **EXAM** → gate Heap.

## Month 3 — interleaving begins
- W9 Heap / Priority Queue (+ interleave earlier patterns). Sun: quiz. (PD101 would activate here in full build.)
- W10 Backtracking. Sun: **EXAM** → gate Graphs.
- W11 Graphs I (traversal, William Fiset). Sun: quiz.
- W12 Graphs II. Sun: **MIDTERM** (cumulative W1–W12, ≥B to enter month-4 depth).

## Month 4 — DP + harder tiers
- W13 1-D DP (Abdul Bari). Sun: quiz.
- W14 1-D DP cont. Sun: **EXAM**.
- W15 2-D DP. Sun: quiz.
- W16 2-D DP cont. Sun: **EXAM**.

## Month 5 — synthesis
- W17 Greedy. Sun: quiz.
- W18 Intervals. Sun: **EXAM**.
- W19 Bit Manipulation. Sun: quiz.
- W20 Mixed hard set + timed drills. Sun: **EXAM**.

## Month 6 — interview simulation
- W21 Timed mixed mocks (2 unseen/day). Sun: quiz.
- W22 Timed mixed mocks + weak-pattern remediation. Sun: **EXAM**.
- W23 Full mock loops, communication focus. Sun: quiz.
- W24 **FINALS** (cumulative, closed-book, timed). Readiness verdict.
