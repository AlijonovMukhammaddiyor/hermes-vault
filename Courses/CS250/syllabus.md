# CS250 — Data Structures & Algorithms (4 credits)

## Outcomes
Solve unseen LeetCode problems under time pressure by pattern, derive time/space complexity,
and communicate the approach as an interviewer expects. Target: FAANG-senior DSA bar.

## Spine
NeetCode 150, worked in **pattern order** (units unlock in this fixed sequence):
1. Arrays & Hashing
2. Two Pointers
3. Sliding Window
4. Stack
5. Binary Search
6. Linked List
7. Trees
8. Tries
9. Heap / Priority Queue
10. Backtracking
11. Graphs
12. 1-D DP
13. 2-D DP
14. Greedy
15. Intervals
16. Bit Manipulation

## Best source per unit
- Pattern intuition + walkthrough: **NeetCode** video for the pattern/problem.
- Theory / proofs: **MIT 6.006**.
- Graphs: **William Fiset**.
- DP: **Abdul Bari**.

## Grade weights
HW 30 · Quiz 20 · Exam 20 · Final 30.

## Rubric (per graded problem)
- Correctness — binary (AC or not).
- Optimal complexity — binary (meets expected big-O or not).
- Pattern note — 3 pt (names the pattern, why it applies, the invariant/trick).
- Communication — 3 pt (clear approach + complexity stated as in an interview).

## Proof of work (HW gate)
**Verified LeetCode AC** (via leetcode MCP) **+ a 3-line pattern note** written to
`Concepts/patterns/<pattern>.md`. No AC or no note ⇒ not proven ⇒ rolls over + debt++.

## Difficulty ramp
Start Easy in each new pattern; ramp Medium as streak rises; interleave earlier patterns from
month 3. Honors (GPA ≥3.7) unlocks Hard tiers earlier.
