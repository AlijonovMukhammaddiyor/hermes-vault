# 📚 Course Catalog — Hermes University

## CS250 — Data Structures & Algorithms  ·  4 credits
*Solve an unseen Medium LeetCode problem by recognizing its pattern, implementing an optimal solution within ~30 minutes, and stating its time/space complexity as an interviewer expects.*

- **Prerequisites:** none
- **Units (11):** Arrays & Hashing → Two Pointers → Stack → Binary Search → Sliding Window → Linked List → Trees → Tries → Backtracking → Heap / Priority Queue → Semester 1 Finals (cumulative)
- **Grading:** hw 30% · quiz 15% · exam 20% · midterm 15% · finals 20%
- **Enroll:** reply `enroll CS250`

## CS270 — AI Agents  ·  2 credits
*Design, evaluate, and ship an LLM agent at a senior bar: pick an architecture deliberately, build eval/observability for it, engineer its context/RAG, and apply it to real Rizo GO work.*

- **Prerequisites:** none
- **Units (5):** Foundations — prompting, structured output, tool/function calling → Workflow Patterns — chaining, routing, parallelization, orchestrator-workers, evaluator-optimizer → Context Engineering — budgeting, tool-result compaction, memory → RAG & Context — naive → advanced → agentic retrieval → Semester 1 Finals (applied + teach-back)
- **Grading:** hw 40% · quiz 30% · finals 30%
- **Enroll:** reply `enroll CS270`

## CS301 — System Design  ·  4 credits
*Drive an unseen senior-level system-design interview: clarify scope, do capacity BOTEC, propose a high-level design, and defend real trade-offs and failure modes under time pressure.*

- **Prerequisites:** none
- **Units (5):** Fundamentals — delivery framework, BOTEC, CAP, storage, caching, consistency → Core Components — LB, queues, DBs, sharding, replication, CDN → Key Technologies — Redis, Kafka, Cassandra, Elasticsearch, API Gateway → Case Studies I — design real systems end-to-end → Semester 1 Finals (cumulative)
- **Grading:** hw 40% · quiz 15% · exam 10% · midterm 15% · finals 20%
- **Enroll:** reply `enroll CS301`

## PD101 — Behavioral  ·  2 credits
*Walk into a senior/staff behavioral loop with 8–10 true, polished STAR-L stories from real Rizo GO work, delivered with structure and credibility and adapted live to any prompt.*

- **Prerequisites:** none
- **Units (3):** Story Mining — extract 8–10 candidate stories from real Rizo GO work → STAR Drafting — shape stories into STAR-L with quantified impact → Semester 1 Checkpoint (delivery + self-critique)
- **Grading:** hw 40% · finals 60%
- **Enroll:** reply `enroll PD101`
