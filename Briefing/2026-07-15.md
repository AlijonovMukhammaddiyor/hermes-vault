> - **[Show HN: Juggler — open-source GUI coding agent by the creator of JUCE](https://github.com/juggler-ai/juggler)** (193pts HN, 260★) — Jules Storer's native-GUI alternative to terminal-based coding agents · File tree, diff viewer, chat, all in a visual interface. The kind of engineering you only see when a CDN needs to agree on state globally.
